About Project

A Bus Ticket Booking System is an application designed to manage bus reservations, passenger details, seat allocation, ticket cancellation, and waiting lists. It helps passengers easily book and manage tickets while allowing the system to maintain accurate records of seat availability. The main purpose of this system is to automate the process of booking bus tickets instead of handling everything manually. The system stores details such as passenger name, seat number, and booking status.

Main features of Project

  • Enter passenger name
  • Book ticket
  • Cancel ticket
  • Waiting list management
  • Move waiting passenger to confirmed seat after cancellation
  • Display confirmed passengers
